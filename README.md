# 응급 비서 서비스
응급 환자 혹은 보호자가 직접 주변 응급실의 실시간 가용 병상 현황을 확인하여, 최대한 빠르게 치료 받을 수 있도록 도와주는 서비스를 기획하였습니다.

# 나의 역할
`Post`(게시글) 도메인에 대하여 작업하였습니다.
